Shell Scripting

Num sistema operacional há três camadas: A de aplicação, do Shell e o Kernel. Em tradução literal, shell é casca enquanto que kernel é núcleo. O kernel do sistema operacional é um programa do computador que age como mediador do acesso às operações em nível de hardware. Já o shell funciona como uma interface intermediária entre o usuário e o restante do sistema operacional.

Há diversas linguagens de shell para os mais diversos sistemas operacionais. O objetivo aqui será de apresentar problemas cuja resolução se dá por meio de script de shell, o que automatiza muito diversas atividades e consultas realizadas por um administrador de sistemas.

As atividades comentadas se tratam de programação em Bourne Again Shell, ou bash.
